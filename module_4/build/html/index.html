app

Flask dashboard for applicant_data analysis.

Serves a Q&A-style web page displaying analysis results from the applicant_data PostgreSQL database. Also provides an endpoint to scrape new data from thegradcafe.com and insert it into the database. LLM-based standardization populates llm_generated_program and llm_generated_university fields for newly pulled data.

app.insert_row(cur: Cursor, row: dict[str, Any]) → bool

Insert a single row into the database with LLM standardization.

Parses dates, runs LLM standardization on the program field, and inserts into the applicants table. Duplicates are skipped via ON CONFLICT (url) DO NOTHING.

Parameters:

  • cur (psycopg.cursor.Cursor) – An open database cursor.

  • row (dict[str, Any]) – A dictionary of scraped applicant data.

Returns:

True if the row was inserted, False if it was a duplicate.

Return type:

bool

app.pull_data() → tuple[Response, int] | Response

Scrape new data from thegradcafe.com until caught up with database.

Scrapes pages one at a time, stopping when a page has all duplicates (meaning we’ve caught up with existing data). No gaps are left. Accepts a JSON body with an optional max_pages field (default 100) as a safety limit.

Returns:

JSON response with pages scraped, entries processed, and new rows inserted, or a tuple of (response, status_code) on error.

Return type:

flask.Response or tuple[flask.Response, int]

llm_standardizer

LLM-based standardization for program and university names.

Uses TinyLlama via llama_cpp to parse and standardize the combined program/university strings from GradCafe data. Falls back to rule-based parsing if LLM output is invalid.

llm_standardizer.standardize(program_text: str) → Dict[str, str]

Standardize a program/university string using the LLM.

Sends the input through TinyLlama with few-shot prompting, then post-processes with canonical mappings and fuzzy matching. Falls back to rule-based parsing if LLM output is invalid.

Parameters:

program_text (str) – Raw program string (e.g., "Computer Science, MIT").

Returns:

A dict with standardized_program and standardized_university keys.

Return type:

dict[str, str]
